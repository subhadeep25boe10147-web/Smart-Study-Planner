# 📚 Smart Study Planner (Python CLI App)

## 📌 Problem
Students often struggle with managing their study time and keeping track of tasks. This leads to missed deadlines and poor productivity.

## 💡 Solution
The Smart Study Planner is a simple command-line application built using Python that helps students:
- Add and manage study tasks
- Mark tasks as completed
- Track progress
- Store data permanently using file handling

## 🚀 Features
- Add new study tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Track overall progress
- Data stored in JSON file

## 🛠 Tech Stack
- Python
- JSON (for data storage)
- File Handling
